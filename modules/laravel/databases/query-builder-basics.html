    # Connecting to a Database

    Laravel simplifies the connection to a database by including the connection out of the box. The configuration setting for the database can be viewed and updated in the environment configuration file.

    To update the database settings, open the .env file in the root directory of your Laravel project, and then find and update the following lines with your specific database settings:

    DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel
DB_USERNAME=root
DB_PASSWORD=

    # Query Builder

    Laravel's query builder is a convenient way to create and run database queries. Built off of the PDO interface, query builder has all the power you would expect, but with an even simpler interface.

    # Adding the DB facade

    Query builder uses the DB facade for many of its commands. Laravel does not make facades available by default, so we do need to add it to the page before we can use it. That is accomplished using the use statement.

    use Illuminate\Support\Facades\DB;

    # Retrieving results

    # Retrieving all rows

    To retrieve results using query builder, you may use the table method. The table method returns an instance of the specified table in which constraints can be applied. The get method is use to retrieve the final results.

    <?php

namespace App\Http\Controllers;

use App\Http\Controllers\Controller;
use Illuminate\Support\Facades\DB;

class UserController extends Controller
{
    /**
     * Show a list of all of the application's users.
     *
     * @return Response
     */
    public function index()
    {
        $users = DB::table('users')->get();

        return view('user.index', ['users' => $users]);
    }
}

    The get method returns a collection containing the results as objects. Each column of each row is a property of the object.

    foreach ($users as $user) {
    echo $user->name;
}

    # Retrieve a single row

    To retrieve just a single row, the first method may be used instead of the the get method. The first method will return a single row object.

    $user = DB::table('users')->where('name', 'John')->first();

echo $user->name;

    To retrieve a single row by its id, use the find method:

    $user = DB::table('users')->find(3);

echo $user->name;
